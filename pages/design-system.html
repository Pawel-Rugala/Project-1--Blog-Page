Please use chrome dev tools to see css properties

HTML:

  • index.html - landing page or home page
  • ./pages - remaining html files for each page

CSS code is split in two folders:

  • pages - if css apply only to specific page
  • shared - if css is shared between pages

    • CSS in shared folder is also split based on functionality:

    • Elements - navbars, headers, sections, footer etc
    • Typography - font related
    • Grid - grid related
